# Plugins

This directory contains all plugins available in this marketplace.

Each plugin should have its own directory with:
- `.claude-plugin/plugin.json` - Plugin manifest
- `README.md` - Plugin documentation
- Plugin implementation files (skills/, commands/, mcp/, etc.)

## Adding a Plugin

See [CONTRIBUTING.md](../CONTRIBUTING.md) for guidelines on adding plugins to this marketplace.

## Available Plugins

Currently, this marketplace contains no plugins. Check back soon or contribute your own!

To see available plugins, run:
```bash
node scripts/list-plugins.js
```
