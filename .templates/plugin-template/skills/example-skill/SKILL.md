# Example Skill

A template skill demonstrating how to create skills for Claude Code.

## What it does

This skill provides an example of how to structure and document a Claude Code skill.

## Usage

```
/example-skill
```

## Parameters

None required for this example.

## Examples

```
/example-skill
```

This will execute the example skill functionality.

## Implementation Notes

Skills are defined using SKILL.md files that describe:
- What the skill does
- How to use it
- Expected parameters
- Example usage

For more information, see the [Claude Code documentation](https://code.claude.com/docs).
