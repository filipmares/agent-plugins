# Utilities

General-purpose utility plugins for file operations, text processing, and system tasks.

## Available Plugins

Currently no plugins in this category. Be the first to contribute!

## Contributing

Want to add a utility plugin? See the main [CONTRIBUTING.md](../../CONTRIBUTING.md) for guidelines.

Utility plugins should:
- Solve a specific, well-defined problem
- Be reusable across different contexts
- Include clear documentation
- Handle errors gracefully
- Be efficient and performant

## Plugin Ideas

Looking for inspiration? Consider creating:
- File operations (copy, move, rename, delete)
- Text processing utilities
- String manipulation tools
- Date/time formatters
- Hash and checksum calculators
- Compression/decompression tools
- Path utilities
- Environment variable managers
- Configuration file parsers
- Template engines
- Regular expression helpers
