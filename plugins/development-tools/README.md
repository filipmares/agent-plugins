# Development Tools

Plugins for code development, formatting, analysis, and build automation.

## Available Plugins

### [JSON Formatter](./json-formatter)
Format, validate, and manipulate JSON data with various options including pretty-printing, minification, and key sorting.

## Contributing

Want to add a development tool plugin? See the main [CONTRIBUTING.md](../../CONTRIBUTING.md) for guidelines.

Development tool plugins should:
- Follow language-specific best practices
- Include comprehensive error handling
- Provide clear output and feedback
- Be well-documented with examples
- Work independently without external services when possible

## Plugin Ideas

Looking for inspiration? Consider creating:
- Code formatters for different languages
- Linters and style checkers
- Code generators and scaffolding tools
- Dependency analyzers
- Build tool wrappers
- Test runners and utilities
- Documentation generators
- Code complexity analyzers
- Refactoring helpers
