# Web Tools

Plugins for web-related functionality including scraping, HTTP requests, and browser automation.

## Available Plugins

### [Web Fetcher](./web-fetcher)
Fetch and parse web content for AI analysis. Simple HTTP/HTTPS client for retrieving web pages and extracting content.

## Contributing

Want to add a web tool plugin? See the main [CONTRIBUTING.md](../../CONTRIBUTING.md) for guidelines.

Web tool plugins should:
- Handle HTTP/HTTPS requests safely
- Include proper error handling
- Respect robots.txt and rate limits
- Validate URLs before making requests
- Handle timeouts gracefully

## Plugin Ideas

Looking for inspiration? Consider creating:
- API client wrappers for popular services
- Web scraping utilities
- RSS/Atom feed readers
- HTTP request builders
- Browser automation tools
- Web authentication helpers
